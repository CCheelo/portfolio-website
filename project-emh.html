EMH Testing — Lusaka Securities Exchange

2025 · Final-Year Research R · Statistical Testing 33-page report

A statistical investigation into whether stock returns on Zambia's Lusaka Securities Exchange follow a random walk — the foundational assumption of weak-form market efficiency. Used multiple complementary tests on four years of daily closing prices.

Full Report (PDF) GitHub Code Summary Overview

Summary Overview

Research Question

Does the Lusaka Securities Exchange (LuSE) follow weak-form market efficiency? In other words, can past prices be used to predict future prices?

Data & Methodology

Daily closing prices of LuSE-listed companies from January 2022 to October 2025. Three complementary tests applied to log returns:

  • → Runs Test — nonparametric check for randomness in return signs
  • → Ljung-Box Test — joint test for autocorrelation across lags
  • → Variance Ratio Test — tests whether return variance scales linearly with horizon

Key Findings

Results revealed partial weak-form inefficiency: several listed stocks showed statistically significant deviations from a pure random walk, indicating that past return patterns carry some predictive information.

The full report includes descriptive statistics, complete test results, autocorrelation function plots, variance ratio test outputs, and stock price trend graphs in the appendices.

Why It Matters

Market efficiency has direct implications for investors and regulators in emerging markets. If markets aren't fully efficient, technical analysis can produce abnormal returns — but it also signals informational frictions worth addressing.
