Summary Overview

The Platform

PRC System is a multi-page web application for managing football academies and their players. The platform includes authentication, an academies dashboard, player listings, individual profile pages, and a data center for awards and competition outcomes.

Tech Stack

  • → Frontend — HTML, CSS, JavaScript, Bootstrap
  • → Backend — Firebase (authentication, Firestore database, storage)
  • → Data — JSON-structured player, academy, and award records

Team Workflow

The team is structured across four collaborating groups: Core Dev (Firebase/backend), Frontend (HTML/CSS/Bootstrap), UI/UX (design system), and Data & Content (JSON structures). We use a feature-branch Git workflow with PRs into dev and a single project lead merging dev into main for stable releases.

My Contribution

Working across the project on collaborative tasks aligned with my assigned files and the team's workflow conventions. Detailed contribution log will be published alongside the live demo.

Status

The project is in active development. The live demo and screenshots will be added here once the platform reaches a stable milestone and any personal academy/player data has been replaced with anonymized examples.
