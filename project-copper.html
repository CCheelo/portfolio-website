Copper Price Forecasting

2024–2025 · Academic Project R · ARMA · Time Series

Time series forecasting of copper prices — a critical input for Zambia's economy — using ARMA models trained on weekly Bank of Zambia data. The project explored stationarity, model selection via AIC/BIC, and out-of-sample forecast accuracy.

Project Scope

Copper accounts for over 70% of Zambia's export earnings, making accurate price forecasts important for fiscal and trade planning. This project applied classical time series methods to weekly copper price data to evaluate how well past prices can predict near-term movements.

Methods Used

  • → Stationarity testing (ADF, KPSS)
  • → ACF/PACF inspection for ARMA order selection
  • → Model selection via AIC and BIC criteria
  • → Out-of-sample backtesting with RMSE and MAE
